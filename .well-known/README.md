# Apple Pay Domain Association

The file `apple-developer-merchantid-domain-association` must be placed in this directory.

## How to get the file
Contact your SagaPay account manager or download it from the SagaPay merchant portal.

## Requirements
- Filename: `apple-developer-merchantid-domain-association` (no extension)
- Content-Type: `text/plain`
- Must be publicly accessible at `https://yourdomain.no/.well-known/apple-developer-merchantid-domain-association`
- No password protection, proxies, or redirects
